# Nisemata
A tiny generative sequencer.
[DEMO](http://squeak.sakura.ne.jp/etc/Nisemata/)


## Usage

```
cd static
python -m SimpleHTTPServer 8080
```

Open in browser [http://localhost:8080](http://localhost:8080)
